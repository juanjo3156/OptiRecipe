Para poder ver los estilos de este proyecto tienes que tener instalado node y npm en tu computadora
Recomiendo la que dice LTS:
https://nodejs.org/en

Después en el cmd o powershell instala npm con el comando:
npm install -g npm

Verifica la version con 
npm --version 

igual aquí hay un video por si te sale un error:
https://www.youtube.com/watch?v=Z-Ofqd2yBCc

después abre el proyecto en visual studio, abre la consola e instala los módulos de node con el comando:
npm install

para empezar a usar los estilos escribe el comando:
gulp dev

y eso es todo.
